# IoT Pick & Place Robotic Vehicle with Surveillance Camera

## Project Overview
This project focuses on the design and development of an IoT-enabled pick and place robotic vehicle capable of automated object handling and remote surveillance. The system is controlled over the internet using the Blynk IoT platform, enabling real-time monitoring and operation.

## Objectives
- To automate pick and place operations
- To reduce human effort in repetitive material handling tasks
- To enable remote control and monitoring using IoT
- To integrate robotics with Industry 4.0 concepts

## Technologies Used
- NodeMCU ESP8266
- Arduino IDE
- Blynk IoT Platform
- DC Motors and L293D Motor Driver
- Robotic Arm and Gripper
- Surveillance Camera

## System Features
- Remote control via mobile application
- Real-time video monitoring
- Automated object pick and place
- Reliable wireless communication

## Applications
- Manufacturing and assembly lines
- Packaging and material handling
- Industrial automation systems
- Smart factories
